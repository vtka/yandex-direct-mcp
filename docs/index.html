Установка

Настройка за 5 минут

Подключите MCP-сервер к Claude Code и начните управлять рекламой

1

Создайте OAuth-приложение

Зарегистрируйте приложение на oauth.yandex.ru с правами доступа к API Яндекс Директа

2

Подайте заявку на доступ к API

В Яндекс Директ перейдите в Инструменты → API → Мои заявки. API начинает работать сразу после подачи

3

Получите OAuth-токен

Авторизуйтесь через OAuth и скопируйте токен из адресной строки

4

Добавьте конфиг

Скопируйте конфигурацию MCP-сервера в .mcp.json вашего проекта или в глобальный ~/.claude.json

5

Готово!

Перезапустите Claude Code — все 30 инструментов доступны через чат

.mcp.json
{
  "mcpServers": {
    "yandex-direct": {
      "command": "ruby",
      "args": ["path/to/bin/server"],
      "env": {
        "YANDEX_DIRECT_TOKEN": "ваш_токен"
      }
    }
  }
}
